Question 2

What are Dan Saffer's four approaches to interaction design - and a short description of each? Which approach suits your software development style best?

  • User Centered Design (UCD) -> Involves collaboraitng, quering and testing with users throughout all steps of the sesign process
  • Activity Centered Design (ACD) -> Based in the tasks or activities users preform with the application, The design process revolves around optimizing these activities to make them as efficient and effective as possible.
  • Systems Design -> Looks at the application design from a broader perspective. It considers the entire system of interactions, including infrastructure, processes, and the relationships between different components of the application.
  • Genius Design -> In this approach designers rely on their own experience to implement the application without user input, the idea here is that the designers know the users best.
    My approach which fits my style the best would be a mesh between Genius Design and User Centered Design, in the fact that I often query the user for some basic requirements and needs and then from there I can flesh out the rest on my own.

Question 3

What is "cooperative design" and how does it fit in with what you know about Agile?

  • Cooperative Design is a design process which aims to actively involve all stakeholders in a project, the goal is to have the users contribute a great deal towards the product they will be using
  • Cooperative Design Relates to Agile because Agile is a methodology focused on flexibility, collaboration, and iterative development, which aligns closely with the principles of cooperative design. Much like Cooperative Design, Agile includes the users feedback and requirements. To integrate Cooperative Design into Agile, users can be influential throughout more than just the sprint planning and requirment idenfication phase.

Question 5

Look at Activity 2.5 and analyze NETFLIX for the seven listed characteristics. (If you have textbook edition 5, the seventh characteristic is satisfaction.)

  • Effectiveness -> Netflix effectively provides a vast library of content, with algorithms that recommend shows and movies based on user preferences. In addition they have fast and seamless video play allowing for an effective service.
  • Efficiency -> Netflix's interface is designed for quick access to content, with intuitive menus and search functions. Load times are often slow but can be an issue with users sometimes.
  • Safety -> Netflix uses secure login processes, encryption, and multi-factor authentication options to protect user accounts.
  • Utility -> Netflix offers a wide array of features, including personalized recommendations, multiple user profiles, content downloading, and parental controls.
  • Learnability -> Netflix is highly learnable, with a straightforward user interface that new users can quickly understand. Even older people can understand the interface and interact with it well.
  • Memorability -> Netflix's interface is consistent and simple, making it easy for users to remember how to navigate the platform, they are simple and intuitive.
  • Satisfaction -> User satisfaction is generally high, with praise for its vast content library, ease of use, and personalized recommendations. However show cancellations and increasing prices bring dissatisfaction.
