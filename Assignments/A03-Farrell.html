1 The conceptional model of the Xerox Star ★ was a very good model, so good infact that its concepts took root in what we see as modern Operating Systems today. Some of the key good concepts were rooted in the familiarity of known actions such as mimicing placing a file in a folder or dragging a document accorss the screen.
2 Instructing - Users issue direct commands, like typing, clicking, or using gestures, to control the system.
Conversing - Users have a dialog with the system through text or voice, receiving responses in kind.
Manipulating - Users interact with objects in virtual or physical spaces by dragging, dropping, or moving them.
Exploring - Users navigate through virtual or physical environments, discovering or interacting with the space.
Responding - The system initiates, and users choose whether to react, such as responding to notifications or prompts.
3 Designer’s Model - The model the designer has of how the system should work
System Image - How the system actually works, which is portrayed to the user through the interface, manuals, help facilities, and so on
User’s Model - How the user understands how the system works
4 Main Concepts & Metaphors - A paperback book 📚 uses physical metaphors like "pages" and "chapters" that are familiar and tactile. An ebook uses similar metaphors (pages, bookmarks) but in a digital form, allowing for navigation through virtual pages.
Differences - An ebook replicates the reading experience but adds functionalities like adjustable font sizes, search capabilities, and digital bookmarking. These features offer convenience but may reduce the tactile pleasure of a physical book.
Paper Informs Digital - Concepts such as pages, chapters, bookmarks, table of contents, and text formatting have inspired digital.
New Functionality - Real-time navigation, traffic updates, voice guidance, location sharing, and route suggestions based on current conditions.
What is Difficult? - Digital Navigation is comparatively harder on digital media compared to holding a physical book where it is all intuitive.
Pros - Portability, Accessibility, Interactive Features, and Cross Device Compatability.
Cons - Loss of Tangibility, Screen Fatigue, and Battery Dependancy.
